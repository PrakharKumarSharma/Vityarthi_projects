# Personal Expense Tracker

A beginner-friendly Python project to track daily expenses.  
Uses simple concepts like lists, functions, loops, and dictionaries.

# Features
- Add, view, and delete expenses
- Manage categories
- Total and category-wise reports
- Highest expense display
- Simple menu-driven interface

# Technologies Used
- Python (core features only)

# How to Run
1. Clone this repository
2. Run `main.py`
3. Follow on-screen instructions

# Project Structure
main.py  
expenses.py  
categories.py  
reports.py
