# Statement

# Problem Statement
People struggle to keep track of daily spending. A simple solution is needed to record expenses, categorize them, and produce summaries.

# Scope
The project includes expense management, category management, and basic reporting.

# Target Users
Students, professionals, or anyone wanting to track daily spending.

# High-Level Features
- Add expenses
- Manage categories
- Total & category-wise reports
- Highest expense tracking
